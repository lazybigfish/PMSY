## 实施进展

### ✅ 已完成

#### 1. 前端核心模块

**文件**: `src/lib/upload/UploadCore.ts`

功能：
- 统一的上传接口 `upload()` 和 `uploadBatch()`
- 文件验证（大小、类型）
- 图片压缩（支持多种预设：avatar/thumbnail/post）
- 上传进度回调
- 取消上传功能
- 统一的错误处理

**文件**: `src/hooks/useUpload.ts`

功能：
- React Hook 封装
- 管理上传状态（isUploading, progress, error, result）
- 支持取消和重置

**文件**: `src/components/upload/ImageUploader.tsx`

功能：
- 图片上传组件
- 点击上传、拖拽上传、粘贴上传
- 图片预览
- 拖拽排序
- 删除功能
- 上传进度显示
- 错误提示

#### 2. 后端API

**文件**: `api-new/src/routes/upload.ts`

接口：
- `POST /api/upload` - 统一上传接口
- `GET /api/upload/config` - 获取上传配置
- `DELETE /api/upload/:fileId` - 删除文件

特性：
- 数据库事务保证（文件存储+记录创建原子操作）
- 自动关联 task_attachments / project_files 表
- 权限控制（上传者或管理员可删除）
- 日志记录

**注册路由**: `api-new/src/index.ts`

```typescript
app.use('/api/upload', uploadRouter);
```

## 新架构使用方式

### 1. 使用 Hook 上传

```typescript
import { useUpload } from '@/hooks/useUpload';

function MyComponent() {
  const { upload, isUploading, progress, error } = useUpload({
    config: {
      maxFileSize: 5 * 1024 * 1024,
      allowedTypes: ['image/jpeg', 'image/png'],
      compress: { maxWidth: 1920, maxHeight: 1920, quality: 0.85 },
    },
    onSuccess: (result) => {
      console.log('上传成功:', result.fileUrl);
    },
    onError: (error) => {
      console.error('上传失败:', error.message);
    },
  });

  const handleFileSelect = async (file: File) => {
    await upload(file, {
      bucket: 'images',
      folder: 'posts',
      metadata: { taskId: 'xxx', projectId: 'yyy' },
    });
  };
}
```

### 2. 使用组件上传

```typescript
import { ImageUploader } from '@/components/upload';

function MyForm() {
  const [imageUrls, setImageUrls] = useState<string[]>([]);

  return (
    <ImageUploader
      value={imageUrls}
      onChange={setImageUrls}
      maxCount={9}
      maxSize={5 * 1024 * 1024}
      compress="post"
      bucket="images"
      folder="posts"
    />
  );
}
```

### 3. 直接使用 UploadCore

```typescript
import { uploadCore } from '@/lib/upload/UploadCore';

const result = await uploadCore.upload(file, {
  bucket: 'files',
  folder: 'documents',
  onProgress: (progress) => {
    console.log(`${progress.percentage}%`);
  },
});
```

## 注意事项

1. 新上传API地址：`POST /api/upload`
2. 文件名生成规则：`{timestamp}_{uuid}_{originalName}.{ext}`
3. 图片压缩默认启用，可通过 `compress` 参数控制
4. 上传需要登录（requireAuth）
5. 文件存储和数据库记录是原子操作
