# 每日热点功能移除记录

## 变更概述
移除水漫金山模块中的"每日热点"功能，该功能原本用于自动抓取和展示互联网热点新闻。

## 删除的文件

### 1. 前端组件
- `src/pages/water/HotNewsTab.tsx` - 每日热点 Tab 页面组件
- `src/pages/system/tabs/HotNewsConfig.tsx` - 热点新闻配置页面

### 2. 类型定义
- `src/types/water.ts` - 移除了 `HotNews` 和 `NewsComment` 接口定义

## 修改的文件

### 1. 系统设置页面
**文件**: `src/pages/system/SystemSettings.tsx`
- 移除 `HotNewsConfig` 组件导入
- 移除"热点配置"标签页
- 移除 `hot-news` 路由渲染逻辑

### 2. 权限服务
**文件**: `api-new/src/services/permissionService.ts`
- 移除 `hot_news` 表的权限配置

### 3. 备份服务
**文件**: `api-new/src/services/backup/BackupService.ts`
- 移除 `hot_news` 表的备份配置

**文件**: `api-new/src/services/backup/RestoreService.ts`
- 移除 `hot_news` 表的恢复配置

### 4. 数据库初始化脚本
**文件**: `api-new/database/init/009_create_water_module.sql`
- 移除 `hot_news` 表的创建语句
- 移除相关索引和触发器

**文件**: `api-new/database/init/017_create_system_tables.sql`
- 移除 `news_comments` 表的创建语句
- 移除相关索引和触发器

### 5. 数据库迁移脚本
**文件**: `api-new/database/migrations/008_create_water_module.sql`
- 移除 `hot_news` 表的创建语句
- 移除相关索引和触发器

**文件**: `api-new/database/migrations/016_create_system_tables.sql`
- 移除 `news_comments` 表的创建语句
- 移除相关索引和触发器

## 保留的说明

以下文件虽然包含 "HotNews" 字样，但属于论坛热门帖子功能，与每日热点新闻功能无关，予以保留：
- `src/pages/Dashboard.tsx` - 引用的是 `dashboard/components/HotNews.tsx`
- `src/pages/dashboard/components/HotNews.tsx` - 展示论坛热门帖子，非每日热点

## 影响范围

- **前端**: 水漫金山模块不再显示"每日热点"Tab
- **系统设置**: 管理员不再看到"热点配置"选项
- **数据库**: 新环境部署时不再创建 `hot_news` 和 `news_comments` 表
- **备份恢复**: 不再处理 `hot_news` 表的数据

## 后续建议

1. 如需清理现有数据库中的数据，可执行以下 SQL：
   ```sql
   -- 删除新闻评论表（依赖 hot_news，先删）
   DROP TABLE IF EXISTS news_comments;
   
   -- 删除热点新闻表
   DROP TABLE IF EXISTS hot_news;
   
   -- 删除系统配置中的热点相关配置
   DELETE FROM system_configs WHERE key IN ('hot_topic_keywords', 'hot_news_fetch_limit');
   ```

2. 检查是否有定时任务在抓取热点新闻，如有需要停止
