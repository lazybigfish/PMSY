# 项目详情添加开工日期和结束日期需求

## 需求描述

在项目详情页面的基本信息区域，添加**开工日期**和**结束日期**两个字段，用于记录项目的实际开始和结束时间。

## 权限控制

- **可编辑角色**：项目经理、系统管理员
- **查看权限**：所有有权限查看项目的用户

## 功能实现

### 1. 数据库变更

**迁移文件**: `api-new/database/migrations/047_add_project_start_end_date.sql`

```sql
-- 添加开工日期字段
ALTER TABLE projects 
ADD COLUMN IF NOT EXISTS start_date DATE;

-- 添加结束日期字段
ALTER TABLE projects 
ADD COLUMN IF NOT EXISTS end_date DATE;

-- 创建索引
CREATE INDEX IF NOT EXISTS idx_projects_start_date ON projects(start_date);
CREATE INDEX IF NOT EXISTS idx_projects_end_date ON projects(end_date);
```

### 2. 类型定义更新

**文件**: `src/types/project.ts`

在 `Project` 接口中添加：
```typescript
start_date: string | null;  // 开工日期
end_date: string | null;    // 结束日期
```

### 3. 前端页面修改

#### ProjectDetail.tsx
- 更新 `handleUpdateProject` 函数，在更新项目中包含 `start_date` 和 `end_date` 字段

#### ProjectOverview.tsx
- 导入 `Calendar` 图标
- 在基本信息区域添加两个新字段的显示和编辑功能
- 非编辑状态：显示日期或"未设置"
- 编辑状态：显示日期选择器

## 界面展示

### 非编辑状态
```
开工日期: 2026-03-01
结束日期: 未设置
```

### 编辑状态
- 显示日期选择器（`<input type="date">`）
- 支持清空日期
