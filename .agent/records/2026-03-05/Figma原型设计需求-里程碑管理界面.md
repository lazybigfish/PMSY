# Figma 原型设计需求 - 里程碑管理界面重构

## 项目信息

| 项目 | 内容 |
|------|------|
| 设计目标 | 里程碑模板管理界面重构 |
| 设计风格 | 现代简洁、专业高效 |
| 色彩基调 | 蓝色系主色（与现有系统一致） |
| 设计日期 | 2026-03-05 |

---

## 一、设计概述

### 1.1 设计目标

将现有的"版本管理"界面重构为"模板库"管理界面，支持：
- 系统模板、公开模板、私有模板的统一管理
- 模板启用/禁用控制
- 模板详情查看与编辑
- 批量操作功能

### 1.2 页面清单

| 序号 | 页面名称 | 说明 |
|------|----------|------|
| 1 | 模板管理列表页 | 主页面，展示所有模板列表 |
| 2 | 新建模板页 | 创建新的系统模板 |
| 3 | 编辑模板页 | 编辑现有模板 |
| 4 | 模板详情弹窗 | 查看模板完整信息 |
| 5 | 删除确认弹窗 | 删除前的二次确认 |

## 三、交互设计

### 3.1 列表页交互

| 交互 | 行为 |
|------|------|
| 点击行 | 展开/收起详情（可选）或打开详情弹窗 |
| 悬停行 | 显示操作按钮组 |
| 点击复选框 | 选中/取消选中，底部显示批量操作栏 |
| 全选复选框 | 选中/取消选中所有行 |
| 搜索输入 | 实时筛选，300ms 防抖 |
| 筛选下拉 | 点击展开，选择后立即筛选 |

### 3.2 新建/编辑页交互

| 交互 | 行为 |
|------|------|
| 添加阶段 | 在列表底部添加新的阶段卡片 |
| 删除阶段 | 确认后删除，下方阶段上移 |
| 添加任务 | 在阶段内添加新的任务项 |
| 删除任务 | 直接删除，无需确认 |
| 拖拽排序 | 支持拖拽调整阶段顺序（可选） |
| 表单验证 | 失去焦点时验证，错误显示红色提示 |

### 3.3 详情弹窗交互

| 交互 | 行为 |
|------|------|
| 点击阶段 | 展开/收起任务列表 |
| 点击编辑 | 关闭弹窗，跳转到编辑页 |
| 点击禁用/启用 | 切换状态，显示成功提示 |
