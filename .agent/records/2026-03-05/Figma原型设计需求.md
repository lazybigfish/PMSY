## 一、设计概述

### 1.1 项目背景
对现有里程碑功能进行全面升级，实现：
1. 延迟初始化机制
2. 模板选择初始化
3. 自定义保存为模板
4. 系统级模板管理

### 1.2 现有默认模板保留说明
**现有的默认里程碑模板将完整保留，作为系统模板库的第一份公开系统模板。**

现有模板包含7个阶段、36个任务：
- **进场前阶段** - 5个任务
- **启动阶段** - 6个任务
- **实施阶段** - 5个任务
- **初验阶段** - 4个任务
- **试运行阶段** - 2个任务
- **终验阶段** - 3个任务
- **运维阶段** - 1个任务

该模板将作为系统模板（is_system = true），所有用户可见且可选择使用。

### 1.2 设计风格要求
- **风格定位**：现代简洁、专业高效的企业级SaaS风格
- **色彩基调**：以蓝色（#3B82F6）为主色调，搭配中性灰色系
- **设计语言**：卡片式布局、圆角设计、柔和阴影
- **交互感受**：流畅、直观、反馈及时

### 1.3 参考现有设计
请查看现有系统页面保持一致性：
- `/src/pages/projects/tabs/Milestones.tsx` - 现有里程碑页面
- `/src/pages/system/tabs/MilestoneTemplates.tsx` - 现有模板管理页面

## 三、交互细节要求

### 3.1 通用交互
- 所有按钮需要有 hover 状态
- 表单输入需要有 focus 状态
- 操作成功/失败需要有 toast 提示
- 加载状态需要有 loading 指示器

### 3.2 特殊交互

**模板选择页面**：
- 点击模板卡片可选中（单选）
- 选中后卡片边框高亮
- 点击"查看详情"打开详情弹窗
- 点击"选择此模板"直接选中并关闭

**保存为模板弹窗**：
- 标签支持输入后回车添加
- 已添加标签可点击删除
- 可见性切换时显示提示说明

**模板管理列表**：
- 支持表头排序
- 支持行内快捷操作
- 批量操作需要确认弹窗
